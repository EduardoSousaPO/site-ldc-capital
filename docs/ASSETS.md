## Regras de uso de logos

1. **Fundo claro** → use `logo-ldc-horizontal.png` ou `logo-ldc-principal.png`
2. **Fundo escuro** → use `logo-ldc-branca.png`
3. **Cards de carrossel Instagram** → use as versões em `carousel/` (já pré-cortadas)
4. **Novos materiais de marca** → use os arquivos `LDC Capital - Logo Final_*.png` como referência master
5. Nunca distorcer proporções — todos os logos têm proporção fixa
